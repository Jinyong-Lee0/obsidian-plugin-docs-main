# EditorSuggestTriggerInfo

## 속성

### start

```ts
start: EditorPosition;
```

트리거 텍스트의 시작 위치입니다. 이는 팝오버를 위치시키는 데 사용됩니다.

### end

```ts
end: EditorPosition;
```

트리거 텍스트의 끝 위치입니다. 이는 팝오버를 위치시키는 데 사용됩니다.

### query

```ts
query: string;
```

제안 콘텐츠를 생성하는 데 사용되는 쿼리 문자열입니다 (일반적으로 start와 end 사이의 텍스트).
