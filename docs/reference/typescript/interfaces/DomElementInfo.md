# DomElementInfo

## 속성

### cls

```ts
cls: string | string[]
```

할당할 클래스입니다. 공백으로 구분된 문자열이나 문자열 배열일 수 있습니다.

### text

```ts
text: string | DocumentFragment;
```

할당할 텍스트 콘텐츠입니다.

### attr

```ts
attr: { [key: string]: string | number | boolean; }
```

추가할 HTML 속성입니다.

### title

```ts
title: string;
```

HTML 타이틀 (호버 툴팁용).

### parent

```ts
parent: Node;
```

할당될 부모 요소입니다.

### value

```ts
value: string;
```

값입니다.

### type

```ts
type: string;
```

타입입니다.

### prepend

```ts
prepend: boolean;
```

선행 여부입니다.

### placeholder

```ts
placeholder: string;
```

플레이스홀더(자리 표시자)입니다.

### href

```ts
href: string;
```
