# Obsidian TypeScript API

:::warning
이 문서는 [Obsidian API](https://github.com/obsidianmd/obsidian-api/blob/master/obsidian.d.ts)에서 생성되었습니다.

문서를 생성하는 데 사용된 [스크립트](https://github.com/marcusolsson/dts-docs)는 아직 상대적으로 충분히 테스트되지 않았습니다. 공식 API와의 오류나 일관성이 없는 부분을 발견하면 알려주세요.
:::
