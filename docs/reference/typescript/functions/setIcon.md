# setIcon

```ts
export function setIcon(parent: HTMLElement, iconId: IconName): void;
```

iconId에서 요소에 SVG를 삽입합니다. 아이콘Id와 연결된 아이콘이 없으면 아무 것도 수행하지 않습니다.

## Parameters

| Parameter | Description                         |
| --------- | ----------------------------------- |
| `parent`  | the HTML element to insert the icon |
| `iconId`  | the icon ID                         |
